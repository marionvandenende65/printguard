De technologieThe technology

Uw ogen zien het niet. De printer des te meer.Your eyes don't see it. The printer feels it all the more.

Een scherm toont pixels als lichtpuntjes — een afwijking van ±10 op 255 is onzichtbaar. Een printer berekent raster-dots per kleurzone. Het channel-split patroon (R↑ G↓ tegelijk) liegt tegen de CMYK-scheiding: kleurverschuiving, vlekken of wazigheid. A screen shows pixels as light points — ±10 deviation out of 255 is invisible. A printer calculates halftone dots per color zone. The channel-split pattern (R↑ G↓ simultaneously) deceives CMYK separation: color shift, blotching or blur.
